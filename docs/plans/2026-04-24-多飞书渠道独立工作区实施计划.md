# 多飞书渠道独立工作区 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 支持单进程内多个飞书机器人实例，每个实例拥有独立 workspace、独立 AgentApplication 和独立会话命名空间，同时保持单 runtime 兼容。

**Architecture:** 通过新增 `channels.instances` 配置描述多个飞书 runtime；Gateway 负责 runtime 级路由；消息对象新增 `channel_instance` 与内部 `thread_key`；会话、队列和归档统一改为使用 `thread_key` 作为内部主键。

**Tech Stack:** Python 3.12, asyncio, dataclasses, SQLite, click, pytest
